## 🚀 기능
- **그래프로 점수 시각화**: 그래프를 이용하여 점수와 평균을 보여줌
- **사용자 친화적인 UI**: 직관적인 디자인으로 쉽게 접근 가능
- **리더보드**: 다른 플레이어들과 점수를 비교할 수 있는 기능
